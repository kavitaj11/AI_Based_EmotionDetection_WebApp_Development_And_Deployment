# Decoded Gita Wisdom (Mobile App Starter)

This is an Expo React Native starter app that turns core Bhagavad Gita teachings into short, practical guidance cards for daily life.

## Features
- Wisdom feed with decoded explanations for selected verses
- Built-in search by verse number/theme/keyword
- Reflection tab with journaling prompts and a simple 10-minute routine
- Clean dark theme UI suitable for Android and iOS

## Run locally
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start Expo:
   ```bash
   npm run start
   ```
3. Open on mobile:
   - Install **Expo Go** on Android/iOS
   - Scan the QR code from terminal/browser
